# Daily Notes

Lightweight scratchpad for capturing quick thoughts, tasks, and notes throughout the day.

## When to Use

**Daily Notes:** Quick capture (Cmd+D in Obsidian)
- Brief meeting jots (before you know if it needs a full page)
- Random thoughts/ideas
- Day-of tasks
- Journal fragments

**Dedicated Meeting Files:** Substantial discussions
- Detailed meeting notes
- Important decisions requiring context
- Meetings you'll reference later

## Template Structure

Each daily note has three sections:

### # Tasks
Quick tasks captured during the day
- [ ] Task 1
- [ ] Task 2

### # Notes
Key observations, insights, meeting mentions
- Note 1
- Note 2

### # Journal
Personal reflections or feelings about the day
- Journal entry

## Integration

Daily notes are automatically read by:
- `/daily-review` - Synthesizes what you captured
- `/daily-plan` - References yesterday's notes for continuity
- `/triage` - Finds orphaned tasks from old notes
- `/week-review` - Detects patterns across the week

## Archival

Daily notes stay in this folder permanently. They're lightweight and useful for pattern detection during weekly/monthly reviews.
