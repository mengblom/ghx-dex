# Meeting Templates - May 8, 2026

These templates are ready for transcript pasting. Each file has a dedicated **Transcript** section at the top for easy copy/paste.

## Workflow

1. **During/After Meeting:** Copy transcript from Teams/Zoom/Granola
2. **Paste:** Replace "_Paste meeting transcript here for later processing_" with the actual transcript
3. **Process:** Run `/process-meetings` to automatically:
   - Extract key points, decisions, action items
   - Update person pages for attendees
   - Link to relevant projects
   - Create tasks from action items
