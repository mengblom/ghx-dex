# Skills#
## Process Inbox ##
Goal: Automatically process all files in "00 - Inbox".

Goals:
- Automatically process all files in "00 - Inbox", including the subfolder '00 - Inbox/Daily Notes/'
- Ensure consistent naming on the format "YYYY-MM-DD hh.mm". If the date is not evident from the content (i.e. meeting time), use the created date from the file.
- Add standardized frontmatter, including topic and tags 
- Categorize the content and move the file to the correct place
- Update the knowledge based on the contents of the file, if relevant. This is especially important for content that reflect my opinions, perspectives etc. 
- Maintain MOCs
- Add links to the file being processed, linking to other MOCs and other relevant content
- Add links to other files, linking to the file being processed if relevant

Some Ground Rules:
- Preserve all original content and author intent
- Ask for clarification on ambiguous categorization or destination
