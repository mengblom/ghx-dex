## Product Overview

### Core Value Props
- Decoupled storage/compute (Snowflake-based)
- No index-based limitations
- Single query language (OPAL) with AI natural language interface
- Knowledge graph for automatic telemetry correlation
- No overages, no throttling

### Pricing Model
- Two SKUs: ingest + compute (not per-product like competitors)
- Significantly lower cost at scale
- Example: Capital One case study showed 20x savings vs DataDog for same features
