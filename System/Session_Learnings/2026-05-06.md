## [09:40] - Day-of-Week Calculation Error in Daily Planning

**What happened:** Generated daily plan for May 6, 2026 labeled as "Tuesday, May 6" when it's actually Wednesday, May 6.

**Why it matters:** Incorrect day-of-week labels cause confusion, especially when today's schedule references "tomorrow's meeting" that is actually today. This error cascaded through the entire plan (Day 2 of 5 should be Day 3 of 5, etc.).

**Root cause:** User's system has "First Day of Week" set to Monday (not Sunday default). The `/daily-plan` skill assumes Sunday-first week numbering, causing an off-by-one error in both weekday names and "Day X of 5" counting.

**User preference:** Monday as first day of week (business calendar standard) - this must be respected in all date calculations.

**Suggested fix:** Update `/daily-plan` skill to use Monday-first week calculations. Added `regional_preferences.first_day_of_week: "monday"` to `System/user-profile.yaml` so skills can reference this preference.

**Status:** preference captured in user profile; skill code needs update to read and respect this setting

## [10:30] - Daily Plans Must Include Full Day Schedule (Past + Future)

**What happened:** Generated daily plans didn't always show complete meeting schedule for the day.

**Why it matters:** User wants daily plan to be "all-encompassing" - showing the complete picture of the day including meetings that already happened AND upcoming meetings.

**User instruction:** "When running /daily-plan, you need to consider meetings that already happened earlier that day. I want the daily plan to be all-encompassing and to contain those meetings as well."

**Correct approach:**
1. Get ALL meetings for today from calendar (regardless of current time)
2. Include both completed and upcoming meetings
3. Optionally mark which meetings are done vs upcoming (e.g., ✓ or [DONE])
4. Show complete day structure so user can see full context

**Example for plan generated at 10:22 AM:**
- ✓ 8:00 AM — EMS MS Portal (completed)
- ✓ 9:05 AM — Staff Meeting (completed)
- 4:00 PM — Thinktiv/Kickdrum Session 1 (upcoming)

**Benefits:**
- Complete picture of the day's schedule
- Context for what's already been accomplished
- Can reference earlier meetings in daily review
- Better understanding of day's shape and energy

**Status:** Update `/daily-plan` skill to fetch and display ALL meetings for the day, marking completed ones
